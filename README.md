# 🛒 ShopMini – E-Commerce Web Application

## 📌 Overview

ShopMini is a modern e-commerce web application that allows users to browse products, add items to cart, and place orders through an interactive and user-friendly interface. The application demonstrates core full-stack concepts including routing, UI rendering, and client-side data handling.

---

## 🚀 Features

* 🔐 User Signup & Login System
* 🛍️ Product Listing with Categories
* 🛒 Add to Cart & Cart Management
* 📦 Order Placement Flow
* ⚡ Dynamic UI Rendering
* 🌙 Clean Dark-Themed Interface
* 💾 Data handled using localStorage

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Templating:** EJS
* **Storage:** localStorage

## ⚙️ Setup Instructions

1. Clone the repository

```id="j4g2p0"
git clone https://github.com/your-username/shopmini.git
```

2. Open project folder

```id="h8y1lc"
cd shopmini
```

3. Install dependencies (for backend)

```id="sk6mva"
npm install
```

4. Run the project

* Open `frontend/pages/index.html` in browser
  **or**
* Use Live Server in VS Code
